# InternSetu 

##  Overview
InternSetu is a full-stack internship management platform that connects students and companies.

It supports secure authentication, role-based access, resume upload, and skill-based internship matching.

The project focuses on real-world backend architecture, API security, and system design, rather than production-level performance optimization.

---

## Features
- JWT-based authentication and authorization  
- Role-based dashboards (Student / Company)  
- Internship posting and application workflow  
- Resume upload and profile management  
- Skill and location-based internship matching  
- Secure, protected REST APIs  

---

## Authentication & Security
- JWT (JSON Web Tokens) for authentication  
- Middleware-based route protection  
- Role-based access control for students and companies  
- Secure token validation for protected endpoints  

---
